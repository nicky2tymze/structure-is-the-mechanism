Seven papers. Ten days. One question: does culture shape AI agent behavior?                                                                                                                                      
  The answer is yes — but not the way anyone expected.                                                                                                                                                             
  Not through metaphor. Not through instruction. Not through personalization. Through structure.

  We built four cultures with identical relational geometry and different surfaces — symbols, nonsense words, plain English, and familiar metaphor. Same shape. Different skin. The behavioral readings were
  identical. The surface didn't matter.

  We tested three external cultures currently in use in the field — OpenAI's personality presets, a real-world personal assistant configuration, and Fabric's prompt framework. None of them moved the structural
  channels above baseline.

  Then we added relational structure to those same external cultures. Same words. Same content. Added geometry. The channels moved.

  Structure prevents degradation. An agent with no culture lost 40% of its structural coherence over an extended session. An agent with structure held steady. Every agent deployed without structure is on the
  degradation curve right now. Nobody is measuring it.

  This paper introduces a diagnostic: two culture-agnostic questions that perfectly separate structured from unstructured cultures. 40 out of 40 measurements. Perfect binary separation. Any culture document can
  be tested in five minutes.

  The field has been using Category 1 and 2 tools and expecting Category 3 results. This paper demonstrates why that expectation fails.

  Structure is the mechanism. Everything else is surface.
